# 📧 Email Classification Assistant using Llama

This project is an AI-powered assistant that classifies emails into categories using a Llama Large Language Model (LLM).  
It helps automatically organize inbox messages by analyzing email content and determining their type or priority.

Large Language Models like Llama can understand natural language and categorize text, making them useful for email management automation. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- Classifies emails using Llama model  
- Automatically analyzes email content  
- Helps organize inbox efficiently  
- Simple command-line based workflow

## 🧠 Technologies Used

- Python  
- Llama (LLM) via Ollama  
- Natural Language Processing (NLP)  
- Prompt Engineering
